Hexo + Github Blog

发表于 2017-08-14

受Qinsman的影响，一直想要部署一个自己的blog，记录一些个人的想法，以及在产品方面的学习与理解。最近也终于使用Hexo 和NexT主题迈出了这一步。由于对前端并不了解，很多想要实现的样式和功能尚在尝试中。这里就正在使用中的功能做记录，以便日后查阅。

1. 首页相关

1.1 新增指定Page

在首页导航栏新增一个指定页面，如，读书笔记、简历、项目
1) 在终端窗口下，定位到Hexo的站点目录，新建页面

2) 编辑刚新建的页面，将页面的类型设置为想要定义的类型

3) 编辑主题配置文件，添加page到menu

此时指定Page已经创建完成，可在导航栏指引。可在该页面md文件中编辑页面内容，如添加post链接、简历内容等。

1.3 背景音乐

在网易云音乐找到需要添加的音乐，然后生成外链，将外链插入至需要的地方，如 /next/layout/_macro/sidebar.swig

1.4 分类数目更新

删除post时，各页面中分类与标签数目并不会自动变化，需将根目录下 db.json 删除，然后重新 hexo s
